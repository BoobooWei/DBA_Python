# Python 基本功进阶版

## 简介

网上学习Python的文章博客非常多，也非常完善，但那毕竟是别人的。只有加入自己的理解和实践并记录才能真的变成自己的。因此才有了这里的一切。

**学习的途径**

1. 兴趣驱动：一定是要实现工作中的问题，不是平白无故来使用python的，例如我需要做数据库自动化和工作管理流程自动化。
2. 系统学习：老男孩的视频
3. 项目学习：GitHub优秀项目的学习模仿
